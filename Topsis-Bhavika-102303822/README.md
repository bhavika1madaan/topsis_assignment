# Topsis-Bhavika-102303822

## Description
This package implements the TOPSIS (Technique for Order Preference by Similarity to Ideal Solution) method.

## Installation
pip install Topsis-Bhavika-102303822

## Usage
topsis data.csv "1,1,1,2" "+,+,-,+" output.csv

## Input Rules
- Minimum 3 columns
- First column: alternatives
- Other columns: numeric only
- Weights and impacts separated by commas
- Impacts must be + or -
